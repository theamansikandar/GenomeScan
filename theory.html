Algorithm Analysis

A comparative study of Classical DFA vs. Quantum Grover search.

Why it matters

Traditional genomic search uses Deterministic Finite Automata (DFA) which processes data sequentially. For large datasets, a classical search takes **O(N)** time—meaning if you double the DNA data, you double the search time.

GenomeScan utilizes **Grover's Algorithm**, which provides a quadratic speedup. By rotating state vectors in a quantum superposition, the complexity is reduced to approximately **Square Root of N**. This allows the system to find gene markers in a fraction of the time required by standard CPU clusters.
